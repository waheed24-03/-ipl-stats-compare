# 🏏 IPL BattleGround: Batsman vs Bowler Duel Analyzer

![Output Screenshot](./OutputSS.png)

This project analyzes IPL match data to compare **batsman vs bowler duels** using real-time statistics. Built with **Python, Pandas, Matplotlib, and Streamlit**, this app visualizes player matchups with:
- Pie chart dominance meter
- Confidence meter
- Strike rate, average, 4s/6s, and dismissal stats

---

## 🚀 Features
- 📊 Compare any batsman and bowler from IPL
- 📈 Pie chart showing dominance
- ⚔️ Duel verdict based on real stats
- 🔥 Confidence meter based on average + SR
- 💡 Simple UI with clean layout

---

## 🛠️ Tech Stack

- `Python`
- `Pandas`
- `Matplotlib`
- `Streamlit`

---

## 📂 Dataset

Uses `deliveries.csv` from Kaggle IPL dataset (you can update with latest seasons).
